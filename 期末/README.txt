# 美股蓝筹动态再平衡投资策略
基于风险平价/固定权重的美股蓝筹资产配置策略，专为长期保值、风险可控设计，适配港澳散户投资需求

## 一、项目概述
本项目实现了**美股蓝筹资产动态再平衡投资策略**，集成风险平价算法、动态调仓机制、风险指标测算与可视化回测功能。
以长期资产保值、严控回撤为核心目标，通过科学的资产配置与风险控制，实现稳健收益。

**回测核心信息**
- 初始本金：100,000 美元
- 回测区间：2020年01月02日 - 2026年04月17日（约6.29年）
- 最终资产：111,060 美元

## 二、核心业绩指标
| 指标 | 数值 | 说明 |
|------|------|------|
| 总报酬率 | 11.06% | 区间累计收益率 |
| 年化报酬率 | 1.70% | 复利年化收益，稳健型 |
| 夏普比率 | 1.32 | 风险调整收益优秀（＞1即为良好） |
| 最大回撤 | -14.8% | 波动控制能力突出 |

## 三、支持策略模式
1. **风险平价+国债对冲**：股票+IEF国债组合，风险均衡配置
2. **纯股票风险平价**：美股蓝筹风险平价策略
3. **固定权重策略**：预设蓝筹股权重，定期再平衡

## 四、核心功能
- 自动获取美股历史行情数据（雅虎财经）
- 基于VIX指数的**动态调仓阈值**（适配市场波动）
- 精细化交易成本计算（佣金+滑点）
- 全维度风险分析（夏普比率、最大回撤、爆仓风险评估）
- 资产收益曲线、权重对比可视化
- 交互式参数调整（支持Google Colab/Jupyter）

## 五、技术栈
- 数据获取：yfinance
- 数值计算：numpy、pandas、scipy
- 可视化：matplotlib、plotly
- 交互界面：ipywidgets
